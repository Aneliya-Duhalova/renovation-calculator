# Тест на iPhone (бързо, безплатно)

За **проба на вашия iPhone** без платен Apple акаунт използвайте **Expo Go**:

1. На iPhone: App Store → инсталирайте **[Expo Go](https://apps.apple.com/app/expo-go/id982107779)**.
2. На Mac в папката на проекта:

```bash
cd renovation-calculator
npm install
npx expo start --tunnel
```

3. Сканирайте **QR кода** от терминала с **камерата на iPhone** (или отворете линка в Expo Go).
4. Mac-ът трябва да е **включен**, докато тествате (сървърът работи на компютъра).

Ако сте на **един Wi‑Fi**, може и без tunnel:

```bash
npx expo start
```

> Това е **тестов режим**, не постоянна инсталация. За икона на екрана без компютър → раздел Android APK по-долу, или iOS TestFlight (платен Apple Developer).

# Постоянна инсталация на няколко телефона

Приложението се инсталира като **обикновено приложение** (икона на екрана), **без Expo Go** и **без компютър** след като веднъж създадете инсталационния файл.

---

## Android (препоръчително – най-лесно за няколко телефона)

### Стъпка 1: Безплатен акаунт в Expo (еднократно)

1. Регистрация: [https://expo.dev/signup](https://expo.dev/signup)
2. На компютъра в папката на проекта:

```bash
cd renovation-calculator
npm install
npx eas-cli login
```

### Стъпка 2: Създаване на APK (еднократно, ~10–20 мин в облака)

```bash
npx eas-cli build --platform android --profile preview
```

- При първи build ще попита дали да създаде проект в Expo – отговорете **Yes**.
- След приключване получавате **линк за сваляне на APK** (и в [expo.dev](https://expo.dev) → вашият проект → Builds).

### Стъпка 3: Инсталация на всеки Android телефон

**Вариант А – директно от линка**

1. Отворете build линка на телефона (Viber, имейл, SMS).
2. Свалете `.apk` файла.
3. При първа инсталация: **Настройки → Сигурност** (или при инсталация) → разрешете **„Инсталиране от неизвестни източници“** за браузъра/Файловете.
4. Отворете APK → **Инсталирай**.

**Вариант Б – един файл за всички**

1. Свалете APK на компютъра.
2. Качете в Google Drive / Telegram / имейл.
3. На всеки телефон свалете същия файл и инсталирайте.

Приложението остава инсталирано. Данните (цени, фирма за оферти) са **на всяко устройство отделно**.

### Обновяване на нова версия

Променете `version` в `app.json`, после:

```bash
npx eas-cli build --platform android --profile preview
```

Раздайте новия APK; на телефоните инсталирайте върху старата версия.

## iPhone (iOS)

За постоянна инсталация **без App Store** са нужни:

- **Apple Developer Program** (~99 USD/година), и
- **TestFlight** (до много тестери) или **Ad Hoc** (ограничен брой устройства по UDID).

```bash
npx eas-cli build --platform ios --profile preview
```

Следвайте указанията на Expo за сертификати. За няколко iPhone най-практично е **TestFlight**: качвате build, поканвате имейли, всеки инсталира от приложението TestFlight.

Без платен Apple акаунт **не може** легално да се раздаде .ipa на произволни телефони.

---

## Алтернатива: Google Play (по желание)

За публично или частно разпространение:

```bash
npx eas-cli build --platform android --profile production
```

В `eas.json` за Play Store сменете `buildType` на `"aab"`. Качвате в [Google Play Console](https://play.google.com/console).

## Често срещани проблеми

| Проблем | Решение |
|--------|---------|
| „Не може да се инсталира“ | Разрешете неизвестни източници; проверете дали APK е за Android (не iPhone). |
| Build чака в опашка | Безплатният план на Expo има опашка; изчакайте или платен план. |
| `Not logged in` | `npx eas-cli login` |
| PDF не се споделя | На някои телефони дайте разрешение за „Файлове“ / „Споделяне“. |
