# Ремонт калкулатор (мобилно приложение)

Мобилно приложение за изчисляване на **квадратни метри** и **стойност на ремонтни дейности** при зададени единични цени.

## Възможности

- Въвеждане на размери на **стени** (ширина × височина в метри)
- Добавяне на **прозорци и врати** – площта им се изважда от общата
- Автоматично изчисление на нетна площ (м²) и периметър за фризове (л.м.)
- Избор на дейности с цена:
  - Гипс картон
  - Шпакловка – 1 ръка / 2 ръце
  - Боя
  - Лепене на фризове (л.м.)
  - Фугиране
  - Чистене на тапети
  - Грундиране
- **Настройка на цени** – запазват се на устройството
- Разбивка и **обща сума** в лева
- **PDF оферта** – генериране и споделяне (Viber, имейл, запазване на файла)

## Стартиране

```bash
cd renovation-calculator
npm install
npm start
```

Сканирайте QR кода с приложението **Expo Go** на телефона (Android / iOS).

### Android / iOS симулатор

```bash
npm run android
npm run ios
```

## Постоянна инсталация на няколко телефона

Вижте **[ИНСТАЛАЦИЯ.md](./ИНСТАЛАЦИЯ.md)** – създаване на **APK** и инсталация на Android без Expo Go.

Кратко:

```bash
npx eas-cli login
npm run build:android
```

След build раздайте APK линка или файла на всеки телефон.

## Пример

- Стена: 4 м × 2.7 м = 10.80 м²  
- Прозорец: 1.2 м × 1.4 м = 1.68 м²  
- **Нетна площ:** 9.12 м²  
- При боя 8 лв./м² → **72.96 лв.** само за боя

Цените по подразбиране са примерни – редактирайте ги от екрана „Цени на дейности“.

## PDF оферта

1. Въведете размери и изберете дейности.
2. Натиснете **„Създай PDF оферта“** под общата сума.
3. Попълнете данни за фирма, клиент и бележки (запазват се за следващ път).
4. **„Генерирай и сподели PDF“** – отваря се системното меню за споделяне.

PDF съдържа: номер и дата на оферта, размери на стени/отвори, нетна площ, таблица с дейности и обща сума.
